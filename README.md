# NoteApp - CPNV

Une application Flask permettant aux étudiants de gérer leurs notes, de calculer leurs moyennes et de suivre leur progression scolaire.

## Fonctionnalités
- Ajout, modification et suppression de notes.
- Attribution de coefficients et de modules.
- Calcul automatique de la moyenne par branche.
- Interface moderne avec **Tailwind CSS** et **daisyUI**.

## User Stories

| ID | En tant que... | Je veux pouvoir... | Afin de... |
| :--- | :--- | :--- | :--- |
| **US1** | Étudiant | Ajouter une note avec un coefficient | Suivre mes résultats d'examens. |
| **US2** | Étudiant | Voir la liste de mes notes par module | Savoir où j'en suis dans chaque branche. |
| **US3** | Étudiant | Calculer ma moyenne générale | Anticiper mon bulletin de fin d'année. |
| **US4** | Étudiant | Modifier une note saisie par erreur | Garder mes données à jour et précises. |git
